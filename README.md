# Setup Notes

```
$ npm i -g yarn
$ git clone https://github.com/AkyunaAkish/dot-env-encryption-practice.git
$ cd dot-env-encryption-practice
$ yarn install
```
# To Run Code

```
$ cd dot-env-encryption-practice
$ yarn test
```

# To Add new NPM Packages

```
$ cd dot-env-encryption-practice
$ yarn add <packageName>
```
